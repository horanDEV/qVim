#  Welcome to qVim!
Requirements:
- python 3.7 or later
- Node.JS
## What is qVim?
qVim - it is vim plugin manager with graphic interface. Here you can find some plugins and qVim install it to your favorite text editor automatically.
## First launch
Execute main.py script using:
```sh
python3 main.py
```
or
```sh
python main.py
```
